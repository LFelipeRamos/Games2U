## Sobre o projeto
### 🎯 Objetivo

A Games 2U é um projeto que busca democratizar o acesso a jogos de tabuleiro, cartas e acessórios, permitindo que mais pessoas possam jogar sem precisar comprar jogos caros ou difíceis de encontrar.

O objetivo do projeto é oferecer uma solução prática e acessível de aluguel e venda de jogos com delivery, levando o entretenimento até o cliente.

### ⚙️ Funcionalidades Principais
-  Catálogo de jogos de tabuleiro e cartas
- Sistema de aluguel por período (dia, fim de semana ou assinatura mensal)
- Pagamento online
- Delivery direto na casa do cliente
- Sistema de devolução via coleta ou pontos parceiros
- Logística com hubs regionais para entregas rápidas

### ⁉️ Como Funciona
1. Escolha o jogo
2. Defina o tempo de aluguel
3. Receba em casa
4. Jogue
5. Devolva

Simples, rápido e acessível.

👥 Público-Alvo

O projeto é voltado para:

Jovens adultos que já conhecem jogos de tabuleiro

Famílias e grupos em busca de lazer acessível

Cafés e bares temáticos

Escolas e espaços educativos

## 🚀 Tecnologias utilizadas

- NextJS
- TypeScript + NodeJS
- SCSS
